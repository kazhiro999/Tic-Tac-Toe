# カスタマイズまわりをデザインするときの考慮ポイント

## 全体カスタマイズ

- 適用画面：kintoneのすべての画面（※ただしシステム設定画面・アプリ設定画面などを除く）
- 設定方法：kintoneシステム管理 > JavaScript/CSSカスタマイズ
- JavaScriptファイルやCSSファイルをアップロードして、kintoneをカスタマイズできる
- PC用とモバイル用でカスタマイズを別々に設定する
  例： https://kintone-sol.cybozu.co.jp/cases/asahi.html

## アプリカスタマイズ

- 適用画面：アプリ画面（※ただしアプリ設定画面を除く）
- 設定方法：アプリの設定画面 > JavaScript/CSSカスタマイズ
- JavaScriptファイルやCSSファイルをアップロードして、kintoneをカスタマイズできる
  PC用とモバイル用でカスタマイズを別々に設定する

## プラグイン

- 適用画面：アプリ画面（※ただしアプリ設定画面を除く）
- 設定方法：アプリの設定画面 > プラグイン
- 自力でJavaScriptやCSSファイルを書かなくても、それらがプラグインにパッケージ化されていてインストールすれば拡張機能が使える
- プラグインを商品として販売している会社もある
- SC（System Consulting）本部が作成しているプラグインもある
