## 概要

CheckboxGroupは、複数の[Checkbox](?path=/docs/components-checkbox--docs)を持つセットグループです。

以下の用途で用いられます：

- CheckboxGroupのなかで、Checkboxは横並びです
- 表示幅が足りなくなったとき、Checkboxを下段に折り返していきます
- 余白はCheckboxとCheckboxの間に、CheckboxGroupが余白を持ちます
