# IconButton

## 概要

アイコンのみのアクションボタンです。

## 構成

<img src={anatomy} alt="IconButton の Anatomy の図" />

- Icon
  - ボタンを押すことで実行されるアクションを示す画像です

## 操作の仕様

### マウス

- Iconをクリックすることで、ボタンに割り当てられたアクションを実行できます
- ホバーすると、アイコンボタンのコンテキストを説明するツールチップが表示される場合もあります
- Iconにホバーすると画像が切り替わる場合もあります

### キーボード

- Tabキーでボタンにフォーカスできます
- フォーカスがある時、エンターキーおよびスぺースキーでボタンに割り当てられたアクションを実行できます
- フォーカスすると画像が切り替わる場合もあります（ホバーと同様の画像です）

### スクリーンリーダー

- フォーカスした時、Iconの代替テキストを読み上げます
- フォーカスがある時、エンターキーおよびスペースキーでボタンに割り当てられたアクションを実行できます

<Notes>ホバー時の画像切り替えの有無などのルールはまだ定まっていません</Notes>

## 見た目の仕様

### カーソル

カーソルはリンクポインターを表示します。

### クリック範囲

Icon部分がクリック可能範囲です。

### ツールチップ

設定されている場合、ブラウザ標準のツールチップが表示されます。

### フォーカスインジケーター

ブラウザ標準のフォーカスインジケーターです。

<Notes>
  {`ただし、フォーカスインジケーターを消していることがあります（hideOutlineがtrueの時）。
  現在確認されているのは1箇所のみ`}
</Notes>
