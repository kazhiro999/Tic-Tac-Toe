### kintone で推奨される Font Size

- 本文：0.875rem (14px)
- アプリ設定画面のページ見出し：1.25rem (20px)
- アプリ設定画面のフィールドラベル：0.875rem (14px)

（参照先）

- [KLOG-1743 「ポータルのデザイン調整（フォントサイズの調整）」](https://bozuman.cybozu.com/k/2963/show#record=1743&tabcode=tab-pg)
- [「タイトル・ラベルのサイズと構成カラー」p.24](https://bozuman.cybozu.com/g/cabinet/view.csp?hid=17412&fid=189006)

#### 参考情報

- Font Size の単位は、em や rem などの相対単位が推奨されています。
- WCAG（Web Content Accessibility Guidelines）における最小 Font Size についての記載はありませんが、<br/>
  様々な Web サービスでは、コンテンツの最小 Font Size を 16px と指定しています。

（参照先）

- [WCAG 1.4.4: Resize Text](https://waic.jp/translations/UNDERSTANDING-WCAG20/visual-audio-contrast-scale.html)
- [Accessible Web](https://accessibleweb.com/question-answer/minimum-font-size/)

### Global tokens

<Notes
  variant={VARIANTS.DANGER}
  children="
現在このデザイントークンは検討中です。
  "
/>

トークン名は `typography.fontSize.{index}` で記述します。例えば `typography.fontSize.1` という名前です。

### Semantic tokens

<Notes
  variant={VARIANTS.DANGER}
  children="
現在このデザイントークンは検討中です。
  "
/>

トークン名は `typography.fontSize.{property}.{modifiers}` で記述します。modifiersは省略可能です。<br/>
例えば `typography.fontSize.button.large` という名前です。

  children="
propertyはコンポーネントに依存しない名前が入ることが望ましいです。

### Global tokens

トークン名は `typography.fontFamily.{lang}` で記述します。例えば `typography.fontFamily.ja` という名前です。

### Semantic tokens

トークン名は `typography.fontFamily.{property}.{lang}` で記述します。例えば `typography.fontFamily.text.ja` という名前です。<br />

トークン名は `typography.lineHeight.{index}` で記述します。例えば `typography.lineHeight.1` という名前です。

### Semantic tokens

トークン名は `typography.lineHeight.{property}.{modifiers}` で記述します。modifiersは省略可能です。<br />
例えば `typography.lineHeight.text` という名前です。
