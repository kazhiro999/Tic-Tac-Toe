# 日本語の記号の用法と英語文言での扱い方

このページでは、kintoneの日本語UI文言で使う記号の用法と英語文言でのそれらの扱い方を記載します。

## 全角小括弧 （）

### 用法

注釈を表す際の区切り記号として使用する。

### 英語での扱い方

日本語の用法に対応する記号は半角小括弧()。

## 半角小括弧 ()

### 用法

基本使用しない。

### 英語での扱い方

N/A

## 鉤括弧 「」

### 用法

用法は以下の4つ。

1. UI画面上の要素を表す<br/>
例：<br/>
「アプリの設定」画面で「アプリを更新」をクリックしてから、再度実行してください。

2. 引用を表す<br/>
例：<br/>
※2022年6月のアップデート時点から記録が開始され、それ以降アクセスがない場合は「-」と表示されます。

3. 特別な用法を示す、物事の呼称を定める、用語を定義する<br/>
例1：<br/>
「アップデートチャネルの選択」機能の導入前と同様のタイミングでアップデートを受け取りたい場合は、[...]<br/>
例2：<br/>
スレッドの作成をスペース参加メンバーに「自分宛」で通知する。

4. 意味のまとまりを表す<br/>
例：<br/>
プラグインを利用すると、kintoneで「できること」が広がります。

### 英語での扱い方

日本語の用法に応じて扱い方が異なる。

1. UI画面上の要素を表す場合<br/>
ダブルクオート（"）に置き換え。<br/>
例：<br/>
Click "Update App" on the app's "App Settings" screen and save the view again.

2. 引用を表す場合<br/>
引用されているのが一文字の場合はシングルクオート（'）、二文字以上の場合はダブルクオート（"）に置き換え。<br/>
例：<br/>
Note: User access is recorded from the time when the June 2022 update takes effect. If a user has not accessed $kintone$ since the update, '-' is displayed.

3. 特別な用法を示す、物事の呼称を定める、用語を定義する場合<br/>
キャピタライズするのみ（特に固有名称の場合）、あるいはダブルクオート（"）［一文字の場合はシングルクオート（'）］に置き換え。<br/>
例1：<br/>
If you want to get feature updates on the schedule equivalent to that before the Update Channel feature was introduced,[...]<br/>
例2：<br/>
Send "To Me" notification about the thread creation to space members

4. 意味のまとまりを表す場合<br/>
記号は使用しない。（英語では記号で囲むという方法で意味のまとまりを表すことはないため。）

## 二重鉤括弧 『』

### 用法

鉤括弧（「」）の中に更に鉤括弧を入れ込みたい場合に使う。

### 英語での扱い方

ダブルクオート（"）の中に更にダブルクオート（"）を入れ込みたい場合、シングルクオート（'）を使う。<br/>
例：<br/>
"Delia said, 'This will never work. ' "

## 半角角括弧 []

### 用法

UI文言のうち、ボタンを表すときに使う。<br/>
例：<br/>
[参照]を押して、ファイル（.zip）を選択してください。

### 英語での扱い方

ダブルクオート（"）に置き換え。<br/>
例：<br/>
Click "Browse" and select a .zip file.

## 半角スラッシュ /

### 用法

基本使用しない。

### 英語での扱い方

N/A

## 全角スラッシュ ／

### 用法

「and」または「or」を示す用途で使う。

### 英語での扱い方

文脈によって「and」か「or」に書き下す。<br/>
「or」の意味で使われており、かつ英文として記号（/）を使う方が明確性、正確性、完結さの観点から良いと判断できた場合は、書き下さずに半角スラッシュを使う。<br/>
例：<br/>
Select a space/thread or click […]

## 中黒 ・

### 用法

用法は以下の3つ。

1. 複数の単語を並列し、まとまった概念を示す<br/>
例：<br/>
「小・中学校、東海道・山陽新幹線」、「食品・雑貨売り場」、「新機能の有効・無効」

2. 外来熟語の単語の区切り<br/>
例：<br/>
ドラッグ・アンド・ドロップ。

3. ビュレット（箇条書き記号）の代わり<br/>
例：<br/>
・スペース<br/>
・スレッド<br/>
・アプリ<br/>

### 英語での扱い方

日本語の用法に応じて扱い方が異なる。

1. 複数の単語を並列し、まとまった概念を示す<br/>
文脈によって「and」か「or」に書き下す。

2. 外来熟語の単語の区切り<br/>
記号は使わない。

3. ビュレット（箇条書き記号）の代わり<br/>
箇条書きのhtmlタグか半角ダッシュ（-）に置き換え。

## 米印 ※

### 用法

用法は以下の2つ。

1. 脚注記号として使う場合<br/>
例：<br/>
kintoneシステム管理者※2、またはcybozu.com共通管理者のみ可能<br/>
※2：スペース内アプリを含むスペースの場合は、スペース内アプリの管理権限が必要です。

2. 注意や補足を表す場合<br/>
例：<br/>
※2022年6月のアップデート時点から記録が開始され、それ以降アクセスがない場合は「-」と表示されます。

### 英語での扱い方

日本語の用法に応じて扱い方が異なる。

1. 脚注記号として使う場合<br/>
脚注記号として使われている場合は（文章中の説明したい箇所に米印がついている場合は）、「\*」を使って訳出する。<br/>
例：<br/>
Allowed only for kintone Administrators\*2 and Users & System Administrators<br/>
\*2: If the space includes in-space apps, the user who performs the action is required to have the app management permission for these apps.

2. 注意や補足を表す場合<br/>
注意や補足を表すために使う場合は（「※」が単体で登場している場合は）、「\*」は使わず、「Note:」、「Note that[...]」、「Tips:」など目的に合った書き方に修正する。<br/>
例：<br/>
Note: User access is recorded from the time when the June 2022 update takes effect. If a user has not accessed $kintone$ since the update, '-' is displayed.
