# IconLink

## 概要

アイコンのみのリンクです。
アイコンリンクはナビゲーション要素として仕様します。
文中などのインラインでは使用しません。常に独立して用いられます。
遷移先がkintoneドメイン内の場合は同一タブで遷移するように使用します。

<img src={anatomy} alt="IconLinkのAnatomyの図" />

- Icon
  - リンクを示す画像です

## 操作の仕様

### マウス

- Iconをクリックすると、リンク先が開きます
- Iconをホバーすると、リンクのコンテキストを表すツールチップが表示されます
- Iconにホバーすると画像が切り替わる場合もあります

### キーボード

- Tabキーでフォーカスできます
- フォーカスすると、画像が切り替わる場合もあります（ホバーと同様の画像です）
- フォーカスがある時、エンターキーでリンク先を開きます

### スクリーンリーダー

- フォーカスした時、Iconの代替テキストを読み上げます
- フォーカスがある時、スペースキーでリンク先を開きます

<Notes>ホバー時の画像切り替えの有無などのルールはまだ定まっていません</Notes>

## 見た目の仕様

### カーソル

カーソルはリンクポインターです。

### クリック範囲

Icon部分がクリック可能範囲です。

### ツールチップ

ホバーするとブラウザ標準のツールチップが表示されます。

### フォーカスインジケーター

ブラウザデフォルトのインジケーターが表示されます。
