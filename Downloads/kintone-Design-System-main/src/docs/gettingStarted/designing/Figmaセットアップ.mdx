## プロジェクト・ファイルの分けかた

「kintone Design Team」teamを基本的には使います。<br/>
（以前までは「Cybozu Design」teamの「kintone Desktop」projectと「kintone Mobile」projectを使っていたので古いデザインファイルはそこに残っています）<br/>
バックログごとにFigmaファイルを分けて管理しています。<br/>
分割チームごとにprojectが分かれているので、依頼のあったチームのプロジェクトにファイルを追加してください。<br/>
バックログ毎のFigmaファイルのテンプレートは各projectに「Template」というファイルがあるのでそれを利用してください。
