# Button

[Figma](https://www.figma.com/file/0EL4UsBtC46KFG7WkKreiYi5/%F0%9F%8E%A8--%F0%9F%92%BB--kintone-UI-Design-Components?node-id=9410%3A69260)

## 概要

ボタンは、ユーザーにアクションを実行してもらう時に使用するコンポーネントです。

- 例：追加、保存、更新、キャンセルなど。

## 構成

<img src={anatomy} alt="Buttonコンポーネントの解剖図" width="500" />

- Container
  - ボタンの形を作る要素です。長方形で構成され、ラベルと背景色を持ちます。
- Label
  - ボタンのテキストラベルです。ボタンを押すことで実行されるアクションを示す文言を指定します。
    Containerの中央に表示されます。

## 種類

### Primary

- 画面上の主要なアクションを促す時に使用します。
- ユーザーに何が重要なアクションかを伝えるために、1画面内での使用頻度は必要最小限にとどめてください。
- ※ 1画面内で必ず使用しなければいけない訳ではありません。

<img src={primary} alt="Primary Buttonの画像です" width="200" />

### Normal

- 汎用性があり、機能に対するアクションやPrimaryボタンの副次的な役割として使用します。
  また、選択肢が複数存在する場合など、並列で使用できます。

<img src={normal} alt="Normal Buttonの画像です" width="200" />

### Delete

- データ削除などの破壊的なアクションに対して、ユーザーに警告する時に使用します
- 警告用途以外は、頻繁に使用しないでください。

<img src={deleteButton} alt="Delete Buttonの画像です" width="200" />

### Save from

- フォームを一時保存する時に使用します。

<img src={saveForm} alt="Save Form Buttonの画像です" width="200" />

## 操作の仕様

### マウス

- Container内をクリックすると、ボタンに割り当てられたアクションを実行します
- Labelが省略表示されている時、ホバーすると全文が表示されます

### キーボード

- Tabキーでボタンにフォーカスできます（disabledの状態を除く）
- フォーカスがある時、エンターキーおよびスぺースキーでボタンに割り当てられたアクションを実行できます

### スクリーンリーダー

- フォーカスした時、ボタンのラベルを読み上げます
- フォーカスがある時、エンターキーおよびスペースキーでボタンに割り当てられたアクションを実行できます

## 見た目の仕様

### 横幅はラベルに対して可変

サイズに応じて指定されている最小幅と最大幅の範囲内で、ボタンの横幅はラベルに応じて可変です。

### ラベルが最大幅を超える場合

最大幅が存在し、ラベルがボタンの最大幅を超える場合、省略記号(…)を文末に表示し、ホバー時にツールチップでラベルの全文字列を表示します。

### カーソル

disabled以外のすべての状態のとき、カーソルはリンクポインターを表示します。

### クリック範囲

Containerの大きさが、クリック可能範囲です。
