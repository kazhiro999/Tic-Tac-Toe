## Global Tokens と Semantic Tokens

### Global Tokens

<Notes
  variant={VARIANTS.DANGER}
  children="
現在デザイントークンは現在整備中です。Figma・実装ともに提供されていません。
  "
/>

- 具体性を持つ「値」を表すトークンです。
- 値に対して 1 対 1 対応する別名をつけます。

トークン名は `{color}.{index}` で記述します。例えば `Red.900` という名前です。<br />
`index` の値は十分に離しておくことが望ましいです。

### Semantic Tokens

- 特定の状態や対象（文字色や背景色など）ごとに「意味」を表すトークンです。
- Global Tokens の別名をつけます。

トークン名は `color.{property}.{colorRole}.{modifiers}` で記述します。例えば `color.background.accent.hover` という名前です。<br />

## Color role

<Notes
  variant={VARIANTS.DANGER}
  children="

  "
/>

Color role は、kintone で用いられる色それぞれに込められた意図を説明しています。

| Color    | Color role |
| -------- | ---------- |
| Blue     | accent     |
| Red      | danger     |
| BlueGray | action     |
| Gray     | neutral    |

## カラーコントラスト

`#FFFFFF` に対して、index が `600` 以上の場合、4.5:1 以上となります。

<Notes
  children="
WCAG AA 規格のコントラスト比を遵守しています（[WCAG 1.4.3](https://www.w3.org/WAI/WCAG21/Understanding/contrast-minimum.html)）

- **大きなテキスト** は、通常のフォントの太さの場合は **18 pt** 以上、太字のフォントの太さは **14 pt** 以上を指します。
  AA の場合、最低基準は 3.0:1 です。
- **標準テキスト** は、通常のフォントの太さの場合は **17 pt** 以上、太字のフォントの太さは **13 pt** 以上を指します。
  AA の場合、最低基準は 4.5:1 です。
- **グラフィックコンポーネント** は、AA の場合、最低基準は 3.0：1 です。
  "
  />
