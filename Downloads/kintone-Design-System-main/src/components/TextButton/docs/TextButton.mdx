# TextButton

[Figma](https://www.figma.com/file/0EL4UsBtC46KFG7WkKreiYi5/%F0%9F%8E%A8--%F0%9F%92%BB--kintone-UI-Design-Components?node-id=9410%3A69260)

## 概要

Normalボタンの次に重要度が低いアクションボタンです。単体でも、他のボタンと組み合わせて使うこともできます。

## 構成

<img src={anatomy} alt="TextButton の Anatomy の図" />

- Label
  - ボタンのテキストラベルです。ボタンを押すことで実行されるアクションを示します。

## 種類

### Normal

- 汎用なアクションに対して使います。

<img src={normal} alt="normalのスクリーンショット" />

### Delete

- データ削除などの破壊的なアクションに対して使います。

<img src={deleteImage} alt="deleteのスクリーンショット" />

## 状態

### Default

- 通常のスタイルです。

<img src={defaultImage} alt="default状態のスクリーンショット" />

### Hover

- マウスカーソルがボタン上に触れた時のスタイルです。

<img src={hover} alt="hover状態のスクリーンショット" />

### Active

- ボタンを押し下げたときのスタイルです。

<img src={active} alt="active状態のスクリーンショット" />

### Focus

Active状態のスタイルに対して、各ブラウザ毎に設定されているアウトラインが表示されます。

## 操作の仕様

### マウス

- ラベル部分をクリックすることで、ボタンに割り当てられたアクションを実行できます

### キーボード

- Tabキーでボタンにフォーカスできます（disabledの状態を除く）
- フォーカスがある時、エンターキーおよびスぺースキーでボタンに割り当てられたアクションを実行できます

### スクリーンリーダー

- フォーカスした時、ボタンのラベルを読み上げます
- フォーカスがある時、エンターキーおよびスペースキーでボタンに割り当てられたアクションを実行できます

## 見た目の仕様

### 横幅はラベルに対して可変

ボタンの横幅はLabelに応じて可変です。

### カーソル

カーソルはリンクポインターを表示します。

### クリック範囲

Label部分がクリック可能範囲です。
