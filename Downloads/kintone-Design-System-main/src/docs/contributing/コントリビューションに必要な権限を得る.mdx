# コントリビューションに必要な権限を得る

## kintone-private org へアカウントを追加する

1. [ITGuide-327](https://bozuman.cybozu.com/k/21893/show#record=327)の手順に従い、GitHubのSSO設定をします。
1. [[公式] IT 窓口](https://bozuman.cybozu.com/k/17545/)に、 kintone-privateへのアカウント追加依頼をします。
   依頼のテンプレートについては、[github.com/kintone-private organization へのアカウント追加](https://bozuman.cybozu.com/k/17545/show#record=34987)を参照してください。
1. IT窓口担当者の指示に従い、[ITGuide-330](https://bozuman.cybozu.com/k/21893/show#record=330)の手順を実施します。

## kintone-Design-System レポジトリの編集可能な権限を得る

1. [Slack の #kintone-design-system_kds チャンネル](https://cybozu.slack.com/archives/C02NWSGMZB8)で@kintone-design-system-team宛にメンションしてkintone-Design-Systemレポジトリの権限が必要な旨をお伝えください。

   雛形：

   ```text
   @kintone-design-system-team
   kintone-Design-System レポジトリの権限付与をお願いします
   github account id: xxx
   ```

1. kintone Design Systemチームの対応が完了次第、編集可能です。連絡をお待ちください。
