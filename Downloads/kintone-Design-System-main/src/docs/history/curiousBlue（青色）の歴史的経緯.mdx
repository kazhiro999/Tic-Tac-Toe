# curiousBlue（青色）の歴史的経緯

## curiousBlueのバージョン

kintoneのcuriousBlueには新旧2つのバージョンがあります。

| | curiousBlue（新）| curiousBlue（旧）|
|---|---|---|
| カラーコード | #0072bf | #3498db |
| 色見本 | <div style={{color: '#ffffff', backgroundColor: '#0072bf', width: '3rem', aspectRatio: '1 / 1'}} /> | <div style={{color: '#ffffff', backgroundColor: '#3498db', width: '3rem', aspectRatio: '1 / 1'}} />
| 文字色として利用される箇所 | React化された新デザイン画面 | Closureの新デザイン画面 |
| アイコン色として利用される箇所 | なし | すべての新デザイン画面 |
| 白色（`#ffffff`）とのコントラスト比 | 5.04: 1 | 3.15 : 1 |

ReactとClosureについては、[画面の呼ばれ方（Closure・React・新旧デザイン）](?path=/docs/history-画面の呼ばれ方（closure・react・新旧デザイン）--docs)を参照してください。

## curiousBlue2つのバージョンがある歴史的経緯

もともと、kintoneではcuriousBlue（旧）を、新デザイン画面の文字色やアイコン色に使っていました。<br />
しかしcuriousBlue（旧）は、白色（`#ffffff`）とのコントラスト比が、WCAGの求める4.5:1を満たしません。<br />
このため、kintoneの新デザイン画面の多くの箇所で、文字色と背景色のコントラスト比がWCAGの基準を満たさない状態でした。

コントラスト比を高めるために導入されたのが、curiousBlue（新）です。<br />
curiousBlue（新）は、白色（`#ffffff`）とのコントラスト比が、WCAGの求める4.5:1を満たします。

今後は、curiousBlue（旧）をcuriousBlue（新）に置き換えていく予定です。<br />
すでに、React化された新デザイン画面の文字色は、curiousBlue（新）に置き換わっています。<br />
しかし、アイコン色はどの新デザイン画面でもcuriousBlue（旧）からcuriousBlue（新）に置き換わっていません。<br />
このためReact化された新デザイン画面において、文字色はcuriousBlue（新）、アイコン色はcuriousBlue（旧）になっています。
