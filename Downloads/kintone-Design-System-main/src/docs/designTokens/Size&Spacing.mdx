## Size

### Global tokens

トークン名は `size.{index}` で記述します。例えば `size.2000x` という名前です。<br />
indexには`0.5rem`を100％とした割合（パーセンテージ）が入ります。

### Semantic tokens

トークン名は `size.{property}.{modifiers}` で記述します。modifiersは省略可能です。<br/>
例えば `size.button.large.minWidth` という名前です。

<Notes
  children="
propertyはコンポーネントに依存しない名前が入ることが望ましいです。
    "
/>

### Tokens Studio for Figma を用いたスタイルの適用方法

Width/Heightにそれぞれ異なるSizeトークンを指定するときには、トークンを右クリックしてWidth/Heightのどちらかを選びます。<br/>
トークンを左クリックで選択すると、Width/Heightの両方に同じトークンが使われるので注意してください。

## Spacing

### Global tokens

トークン名は `spacing.{index}` で記述します。例えば `spacing.200x` という名前です。<br />
indexには`0.5rem`を100％とした割合（パーセンテージ）が入ります。

### Semantic tokens

トークン名は `spacing.{property}.{modifiers}` で記述します。modifiersは省略可能です。<br/>
例えば `spacing.button.large` という名前です。
