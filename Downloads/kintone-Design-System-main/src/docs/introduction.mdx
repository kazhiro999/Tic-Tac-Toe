## kintone Design Systemに含まれるもの

以下の一覧は、画面左側のツリーと対応しています。

- GettingStarted
  - kintone Design Systemを利用するチームが参照する文章一式です。
- Contributing
  - kintone Design Systemにコントリビュートするときに参照する文章一式です。
- Guidelines
  - kintone全体のデザインの指針をまとめた文章一式です。
- History
  - kintoneのデザインの歴史的経緯をまとめた文書一式です。
- DesignTokens
  - kintoneの色や形など、デザインの最小単位であるデザイントークン集です。
  - 実装、Figma、文章の形式で提供しています。形式が異なるだけですべて同じものを指しています。
- kintone-ui (Components)
  - kintoneのさまざまな画面で使われている共通コンポーネント集です。
  - Reactによる実装、Figma、文章の形式で提供しています。形式が異なるだけですべて同じものを指しています。
  - [参考: kintone-ui とは？](?path=/docs/kintone-uiとは？--docs)
